  **Desktop**

  - **Scroll:** zoom in/out
  - **Middle drag:** pan camera
  - **Left drag:** adjust active parameter (Julia C / Mandelbrot z0)
  - **Right drag:** blend mode (X) and exponent (Y)
  - **Double-click:** release manual C / z0 control
  - **Space:** pause/resume animation

  **Mobile**

  - **1 finger:** same as left drag
  - **2 fingers:** same as right drag
  - **Pinch:** zoom

## Overview

A small web-based fractal viewer with the ability to shift the fractal shape by clicking and dragging.
