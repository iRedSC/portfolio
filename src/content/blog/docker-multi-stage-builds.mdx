## Why multi-stage builds matter

Multi-stage builds let you separate build tooling from runtime artifacts.

## A simple pattern

```bash
docker build -t app:latest .
```

## Common pitfalls

- Forgetting to copy only the final build output.
- Leaving debug tools in the runtime layer.
